UserSettings¶

Portal module
UserSettings extension¶
UserSettings/Get¶

Gets settings for current user.

Parameters:clientGUID – GUID of the client for which to get settings
Returns:The settings for the specified client for the current user
Return type:UserSettings
UserSettings/Set¶

Sets settings for current user.

Parameters:
  • clientGUID – GUID of the client for which to set settings
  • settings (string) – the new settings
Returns:

The number of affected rows in the database
UserSettings/Delete¶

Delete settings for a specific client for the current user.

Parameters:clientGUID – GUID of the client for which to delete settings
Returns:The number of affected rows in the database
